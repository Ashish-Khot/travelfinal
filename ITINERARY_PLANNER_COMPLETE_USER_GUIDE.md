# 📋 ITINERARY PLANNER - COMPLETE USER GUIDE

## 🎯 WHAT IS AN ITINERARY PLANNER?

It's a **smart trip planning tool** that helps you:
- 📍 Plan daily activities for your trip
- 🤖 Get AI suggestions for things to do
- 💰 Track your budget by category
- 🗺️ See activities on an interactive map
- ⏰ Organize activities hour-by-hour
- 🌤️ Get weather-based recommendations
- 📥 Export your plan (PDF, HTML, Calendar)

---

## 🚀 QUICK START - THREE WAYS TO CREATE AN ITINERARY

### **METHOD 1: Let AI Generate Everything (Easiest)**

**Step 1: Click "New Itinerary" Button**
```
Location: Top-right corner of Itinerary Planner page
A dialog box will pop up asking for your preferences
```

**Step 2: Fill in Your Preferences**
```
Destination:        "Paris"  or  "Tokyo"  or  "Bali"
Number of Days:     3, 5, 7, 10, etc
Total Budget:       $500, $1000, $2000, etc
Number of People:   1, 2, 3, 4, etc
Travel Style:       Solo, Couple, Family, Friends
Interests:          Museums, Food, Shopping, Adventure, etc
Start Date:         Pick a date from calendar
```

**Step 3: Click "Generate with AI" Button**
- 🤖 AI will create a **complete travel plan**
- ✈️ It will add activities for **each day**
- 💰 It will **balance your budget**
- 🌤️ It will check **weather** and suggest appropriate activities

**Step 4: You'll Get:**
- ✅ Full 7-day itinerary (or however many days you chose)
- ✅ Activities organized by day
- ✅ Time slots for each activity
- ✅ Budget breakdown
- ✅ Suggestions on the map

## 🎨 THE 5 TABS - WHAT EACH DOES

### **TAB 1: MAP VIEW** 🗺️

**What You See:**
- Interactive map showing your destination
- Each activity appears as a colored marker
- Activities grouped by day (Day 1 = Blue, Day 2 = Green, etc)
- Clicking a marker shows activity details

**What You Can Do:**
```
1. DRAG activities on the map to reorder them
   → Optimal travel route between activities
   
2. CLICK on a marker to see:
   - Activity name
   - Time
   - Cost
   - Category
   - Edit button
   - Delete button

3. ZOOM in/out to see activities better

4. Useful For: Seeing your physical route, avoiding overlaps
```

**Example:**
```
Day 1 Activities on Map:
  🔵 9:00 AM - Hotel check-in
  🔵 10:30 AM - Eiffel Tower
  🔵 1:00 PM - Lunch
  🔵 3:00 PM - Louvre Museum
  🔵 7:00 PM - Dinner

You can drag to reorder if any activity is in wrong order
```

---

### **TAB 2: TIMELINE** ⏰

**What You See:**
- Hour-by-hour schedule for each day
- Visual timeline blocks for activities
- Shows duration of each activity
- Color-coded by category

**What You Can Do:**
```
1. DRAG activity blocks to change time
   → If activity runs too late, move it earlier
   → If overlap, adjust times

2. See the whole day at a glance:
   9:00 AM  [========== Eiffel Tower ==========]
   1:00 PM  [=== Lunch ===]
   3:00 PM  [====== Louvre Museum ======]
   7:00 PM  [== Dinner ==]

3. Adjust duration by resizing blocks

4. Useful For: Time management, avoiding schedule conflicts
```

---

### **TAB 3: ACTIVITIES** 📋

**What You See:**
- Complete list of all activities
- Activities grouped by day
- Each activity shows:
  - Name
  - Time (9:00 AM - 11:00 AM)
  - Category (with color badge)
  - Cost
  - Importance (Must-do, Recommended, Optional)

**What You Can Do:**
```
1. ADD NEW ACTIVITY
   Click: "Add Activity" button
   Fill: All activity details
   Save: Saves automatically

2. EDIT ACTIVITY
   Click: Edit icon (pencil) next to activity
   Change: Any detail you want
   Save: Click "Update Activity"

3. DELETE ACTIVITY
   Click: Delete icon (trash) next to activity
   Confirm: Click "Yes, Delete"
   Effect: Removes from map, timeline, budget

4. Example activity list:
   ┌─────────────────────────────────┐
   │ DAY 1                           │
   ├─────────────────────────────────┤
   │ 🔵 9:00-11:00 Eiffel Tower     │
   │    Cost: $20                    │
   │    🗑️ Edit  🗑️ Delete          │
   │                                 │
   │ 🔴 1:00-2:00 Lunch             │
   │    Cost: $15                    │
   │    🗑️ Edit  🗑️ Delete          │
   └─────────────────────────────────┘
```

**What You See:**
- Pie chart: Your total budget breakdown

  ```
  Accommodation:  $300
  Transportation: $150

  Food:           $250

  TOTAL:          $1200

  ```

**What You Can Do:**
```
1. See how much you're spending per category
2. Track if you're over budget

4. Plan which days to spend more/less
5. Export as chart (PDF/Image)

Example:
Total Budget: $2000

If Activities are too expensive:
→ Go back to Activities tab
→ Remove or reduce expensive activities
→ Budget chart updates automatically
```

---

### **TAB 5: AI PLANNER** 🤖

**What You See:**
- AI suggestion controls
- Weather recommendation section
- Optimization button

**Features:**

#### A) **GET ACTIVITY SUGGESTIONS**
```
Step 1: Select which day: "Day 2"
Step 2: Click "Get Suggestions" button
Step 3: AI shows suggestions like:
        - Colosseum
        - Roman Forum
        - Vatican Museums
        - Pasta making class
        - Opera house tour
        etc.
Step 4: Click any suggestion to ADD IT
Step 5: Configure time yourself
```

#### B) **WEATHER RECOMMENDATIONS**
```
What it does:
- Checks weather forecast for your destination
- Suggests activities based on weather

Example:
Day 1: Sunny ☀️
  Suggested: Outdoor sightseeing, parks, beaches
  Avoid: Indoor museums in good weather

Day 2: Rainy 🌧️
  Suggested: Museums, indoor shopping, restaurants
  Avoid: Outdoor hiking, beaches

Day 3: Partly Cloudy ⛅
  Suggested: Mix of indoor/outdoor
  Balance: Museums in morning, parks in afternoon
```

#### C) **OPTIMIZE SCHEDULE**
```
What it does:
- Reorders activities to minimize travel time
- Calculates best route between activities
- Avoids backtracking

Example:
BEFORE:
  Eiffel Tower → Louvre → Sacré-Cœur → Arc de Triomphe
  (Lots of back and forth, inefficient)

AFTER OPTIMIZATION:
  Arc de Triomphe → Sacré-Cœur → Eiffel Tower → Louvre
  (Logical route, less travel time)
```

## 💡 PRO TIPS

1. **Start with AI, then customize** - Don't create from scratch
2. **Use Timeline for time conflicts** - Easier to see overlaps
3. **Check Budget daily** - Don't exceed total limit
4. **Weather matters** - Let AI suggest based on weather
5. **Leave buffer time** - Add 30 min between activities
6. **Research actual costs** - AI is approximate
7. **Share early** - Get feedback before trip
8. **Export often** - Save as PDF as backup
9. **Optimize before finalizing** - Let AI find best route
10. **Review day before** - Confirm activity locations/times
